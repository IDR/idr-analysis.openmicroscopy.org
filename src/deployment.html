Deployment

The VAE runs on a Kubernetes Cluster hosted on an OpenStack cloud at EMBL-EBI.

Kubernetes cluster

A Kubernetes cluster is required to deploy the VAE. At present this is deployed using Ansible and the Kubespray project. See the IDR/deployment repository for the required playbooks.

JupyterHub

The VAE deployment uses the Zero to JupyterHub Helm chart to deploy JupyterHub and configure notebooks. This is an automated process using GitLab CI/CD, and is controlled by the IDR/k8s-analysis-deploy repository using Helmfile.
