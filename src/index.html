Analysis Platform

Welcome to the Image Data Resource Analysis Platform.

The IDR Analysis Platform supports the open and reproducible analysis of data in the IDR. It is built on JupyterHub and is available to anyone interested in exploring and mining the immense range of image data and metadata in the IDR.

All notebook sessions are subject to available resources, are limited in the network resources they can access, and will automatically terminate after a period of inactivity. Do not store anything you wish to keep.

Public access

We have a set of Jupyter Notebooks that can be run by anyone without a login.

GitHub login

If you are interested in developing your own data analysis notebooks on our analysis platform please contact us with your GitHub username. This will give you more computational resources including scratch storage space than is available through the public access.

Elixir AAI login

If you have an Elixir AAI account you can use it to access the analysis platform. This is currently a demonstrator, and is identical to the public access. Please note we are unable to investigate authentication errors between your chosen identity provider and Elixir AAI.
